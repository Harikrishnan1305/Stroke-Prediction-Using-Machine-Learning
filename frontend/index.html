Key Features

Everything you need for comprehensive stroke risk assessment

ML Prediction

Random Forest model trained on 9 clinical parameters including blood pressure, heart rate, cholesterol, BMI, and lifestyle factors.

MRI Brain Scan Analysis

EfficientNetB0 deep learning model analyses brain MRI scans to detect stroke patterns with transfer learning from ImageNet.

Grad-CAM Explainability

AI-generated heatmaps showing exactly which regions of the brain MRI the model focused on to make its prediction.

Combined AI Scoring

Weighted combination of ML (40%) and DL (60%) predictions for more accurate risk assessment when MRI data is available.

Clinical Recommendations

Automatic generation of personalized medical recommendations based on patient parameters and predicted risk level.

Analytics Dashboard

Real-time statistics, risk distribution charts, patient history tracking, and comprehensive prediction analytics.

About This Project

This Stroke Prediction System is an academic and research project that demonstrates the combined use of Machine Learning and Deep Learning for clinical decision support in stroke risk assessment.

The system uses a trained Random Forest classifier on tabular patient data and an EfficientNetB0 CNN with transfer learning on brain MRI images. Both models' predictions are combined using a weighted averaging approach to provide a comprehensive risk score.

Disclaimer: This is a research/educational tool and should NOT be used for actual medical diagnosis. Always consult qualified healthcare professionals.
